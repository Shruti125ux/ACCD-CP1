Creative Prototyping

Interactive Mall Kiosk

Shruti Amritkar • ArtCenter • Fall 2025

Final Prototype

An interactive Rock Paper Scissors game designed as a mall kiosk experience. Users interact with the kiosk using hand gestures detected through ML5.js handpose tracking.

Initial Explorations

Early concept development and wireframing to establish the core interaction model and visual direction for the kiosk interface.

Format Changes

Small changes in UI and color scheme were made during this phase. Camera settings and size of the canvas were adjusted to optimize the user experience and ensure proper hand tracking functionality.

Refining the Interface

Iterative refinement of visual design, layout structure, and interaction patterns to create a polished, user-friendly kiosk experience.

Game Settings

Editing the scoreboard and game logic settings to ensure fair gameplay, accurate score tracking, and smooth state transitions.
