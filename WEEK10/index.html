Project Overview

Transforming the boring Netflix loading screen into something interesting and engaging. This project explores creative animation techniques using P5.js, focusing on time-based functions and oscillation to create a captivating loading experience.

Inspiration

The starting point: Netflix's standard loading screen. Simple, functional, but lacking personality. The goal was to reimagine this experience with dynamic motion and visual interest.

Original Netflix loading screen - the inspiration for transformation

Design Process

1

Initial Explorations - Circular Patterns & Arcs

Started with basic explorations using circular patterns and arc geometry. Experimented with different configurations to understand the visual rhythm and movement possibilities.

First circular pattern exploration
Arc geometry experimentation
Refining circular motion patterns
2

Canvas Optimization - Screen Proportions

Changed the background from square to rectangle to better resemble laptop or TV horizontal screens. This adjustment improved the context and viewing experience.

Horizontal canvas matching TV/laptop screen proportions
3

Layered Arcs - HALF_PI Rotation

Added 3 layers of arcs using numArcs syntax and implemented HALF_PI function for rotation. This created depth and visual complexity through layering.

Three-layer arc system with HALF_PI rotation
4

Rotation Refinement - QUARTER_PI Function

The HALF_PI rotation didn't achieve the desired effect. Switched to QUARTER_PI function for a more subtle and controlled rotation pattern.

Improved rotation using QUARTER_PI function
5

Maximum Detail - 10 Arcs with Netflix Color Theme

Increased total arcs to 10 and reduced thickness to create delicate, refined lines. Applied color gradation from light to dark red to match Netflix's iconic branding.

10 thin arcs with Netflix-inspired red color gradation
6

Color & Shape Exploration - Triangular Forms

Experimented with alternative color schemes and replaced rotating pie shapes with triangles. Initial attempts resulted in overlapping forms that needed refinement.

Color exploration phase - testing different palettes
Triangular rotation - initial overlapping issue
7

Hourglass Concept - Refined Geometry

Modified triangles to create an hourglass shape. Reduced stroke thickness and changed from grey to black for stronger contrast and visual impact.

Hourglass-inspired geometry with refined strokes
8

Dynamic Element - Oscillating Ball

Added a small red ball with back-and-forth oscillating motion, creating the illusion of movement inside an hourglass. This brought life and rhythm to the static forms.

Red ball with oscillating motion inside hourglass forms
9

Surprise Element - Holiday Theme

Added a festive surprise element to break the monotony of continuous loading animation. Incorporated holiday/Christmas aesthetics to give Netflix a seasonal feel.

Holiday-themed surprise element for seasonal Netflix branding

Reflection

This iterative design process showcased the power of experimentation in creative coding. Each modification brought new insights into motion design, oscillation principles, and brand-aligned animation. The journey from basic circular patterns to a sophisticated, themed loading experience demonstrates the value of continuous refinement and creative exploration in interaction design.
